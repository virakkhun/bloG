### Description

bloG is a simple blog website, yet it is minimal.

- Clone this repository

  ```bash 
    git clone https://github.com/virakkhun/weather-app.git
  ```

- Go to the project directory

  ```bash 
    cd weather-app
  ```

- In the project directory

  ```bash 
    yarn install
  ```

- Run on local

  ```bash 
    yarn dev
  ```
